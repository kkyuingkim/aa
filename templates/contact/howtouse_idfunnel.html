아이디퍼널(IdFunnel) 사용방법


'아이디퍼널'은 '쇼핑몰파인더', 'SNS파인더'를 통해 획득한 아이디들을 포털, SNS, 쇼핑몰의 리얼아이디로 존재하는지를 검증해 주는 툴입니다.


1. 아이디퍼널(IdFunnel) 설치


차가운 홈페이지(http://chagaun.net)에서 화면 상단에 있는 '다운로드 -> 아이디퍼널'를 선택한 후 나오는 페이지에서 '다운로드'를 선택하여 설치를 합니다. 
설치를 하게 되면 바탕화면에 '아이디퍼널'라는 아이콘이 바탕화면에 생성 됩니다.






설치환경 : Windows 7 이상에서 테스트가 되었으며 크롬브라우저가 설치가 되어 있어야 합니다. 


설치파일 : IdFunnelSetup.exe














2. 아이디퍼널 실행


바탕 화면에 있는 '아이디퍼널' 아이콘을 더블 클릭하여 프로그램을 실행합니다.




신규로 업데이트될 파일이 있는지 확인하여 패치될 파일이 있는 경우 자동으로 패치가 이루어진 후 '아이디퍼널' 메인 화면이 보여집니다.





3. 로그인



'아이디퍼널' 메인 화면에서 로그인 버튼을 클릭한 후 나오는 로그인 인증 창에서 이메일 아이디와 비번을 입력하여 로그인 을 합니다. 

로그인 아이디가 없는 경우 차가운 홈페이지(http://chagaun.net)에서 회원 가입을 한 후 사용할 수 있습니다. 



로그인에 성공한 경우 화면 하단에 회원님의 만기일이 표시가 됩니다. 아이디퍼널를 첫 로그인 시 자동으로 1일 이용권이 부여됩니다. 

만기일을 연장하고자 하는 경우에는 차가운 홈페이지(http://chagaun.net)에서 '내정보/구매' 코너를 통해 이용권을 구매하는 경우 해당 기간만큼 자동으로 만기일이 연장됩니다.





4. 아이디+도메인 조합



검증하고자 하는 아이디들을 '아이디' 코너에 입력하고 해당 아이디와 결합 할 도메인들을 '도메인' 코너에 입력합니다. 



'아이디+도메인 조합' 버튼을 클릭하면 입력한 각각의 아이디와 도메인을 결합하여 '아이디+도메인 결과' 코너에 보여줍니다. 






5. 검증 시작


메인 화면의 '아이디+도메인 결과' 코너에 검증하고자 하는 이메일아이디들을 입력합니다.

메인 화면의 '사이트리스트'에서 사이트명을 선택하여 검증할 사이트를 지정합니다. 

'검증 시작'을 클릭하면 이메일아이디들을 가지고 지정한 사이트에서 검증을 시작합니다.



지정된 사이트에 해당 이메일아이디가 존재하는 것으로 판단이 되는 경우 '검증 결과' 코너에 각 사이트별로 보여줍니다.





6. 검증된 결과 목록의 활용


추출된 이메일아이디 목록들은 따로 저장하여 이메일 마케팅으로 활용할 수 있습니다. 

또한 구글, 페이스북, 인스타그램 등의 마춤타겟 광고로 활용하실 수도 있습니다.
