SNS파인더(SnsFinder) 사용방법


'SNS파인더'는 블로그, 카페, 쇼핑몰과 같은 사이트에서 관심을 가지고 있거나 구매 의향을 가지고 있는 상품 키워드를 통해 
그와 관련된 블로그, 카페, 쇼핑몰 아이디들을 추천 받아 마케팅에 활용할 수 있도록 지원하는 툴입니다.


1. SNS파인더(SnsFinder) 설치


차가운 홈페이지(http://chagaun.net)에서 화면 상단에 있는 '다운로드 -> SNS파인더'를 선택한 후 나오는 페이지에서 '다운로드'를 선택하여 설치를 합니다. 
설치를 하게 되면 바탕화면에 'SNS파인더'라는 아이콘이 바탕화면에 생성 됩니다.






설치환경 : Windows 7 이상에서 테스트가 되었으며 크롬브라우저가 설치가 되어 있어야 합니다. 


설치파일 : SnsFinderSetup.exe














2. SNS파인더 실행


바탕 화면에 있는 'SNS파인더' 아이콘을 더블 클릭하여 프로그램을 실행합니다. 




신규로 업데이트될 파일이 있는지 확인하여 패치될 파일이 있는 경우 자동으로 패치가 이루어진 후 'SNS파인더' 메인 화면이 보여집니다.





3. 로그인



'SNS파인더' 메인 화면에서 로그인 버튼을 클릭한 후 나오는 로그인 인증 창에서 이메일 아이디와 비번을 입력하여 로그인 을 합니다. 

로그인 아이디가 없는 경우 차가운 홈페이지(http://chagaun.net)에서 회원 가입을 한 후 사용할 수 있습니다. 



로그인에 성공한 경우 화면 하단에 회원님의 만기일이 표시가 됩니다. SNS파인더를 첫 로그인 시 자동으로 1일 이용권이 부여됩니다.

만기일을 연장하고자 하는 경우에는 차가운 홈페이지(http://chagaun.net)에서 '내정보/구매' 코너를 통해 이용권을 구매하는 경우 해당 기간만큼 자동으로 만기일이 연장됩니다. 





4. 키워드 추천 시작



관심을 가지고 있거나 구매하고자 하는 상품을 메인 화면의 '상품 키워드' 란에 입력을 합니다. 

메인 화면의 '사이트리스트'에서 사이트명을 선택하여 검색할 사이트를 지정합니다. 

검색을 하기전에 재외할 키워드가 있는 경우 '재외 키워드'란에 입력을 하면 해당 키워드는 재외가 됩니다. 



'키워드 추천 시작'을 클릭하면 입력된 키워드와 연관된 키워드들을 추출하여 '키워드 추천 결과' 코너에 결과를 보여 줍니다. 






5. 아이디 추천 시작


메인 화면의 '키워드 추천 결과' 코너에 검색하고자 하는 키워드를 입력합니다.


메인 화면의 '사이트리스트'에서 사이트명을 선택하여 검색할 사이트를 지정합니다. 


'아이디 추천 시작'을 클릭하면 입력된 키워드들을 가지고 해당 사이트에서 검색을 시작합니다.



-. 해당 키워드를 포함하는 글이나 사진, 영상을 올린 블로그, 카페, 쇼핑몰 등의 아이디 들을 추출합니다. 



-. 차가운 AI 알고리즘에 의해 구매 의향을 보였다고 판단이 되는 아이디들을 추출하여 해당 사이트 카테고리에 보여줍니다. 

-. 추출된 아이디들이 구글, 페이스북, 인스타그램에서도 사용이 되고 있는지를 확인하여 사용이 되고 있으면 해당 사이트 카테고리에 보여줍니다.





6. 추천된 결과 목록의 활용


추출된 아이디 목록들은 따로 저장하여 이메일 마케팅으로 활용할 수 있습니다. 

또한 아이디 퍼널을 통해 포털, 쇼핑몰, SNS의 실사용자 여부를 검증할 수도 있습니다.
